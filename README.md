# Smart Appliance Assistant

A real-time multimodal AI system that combines custom-trained object detection, large language model inference, and voice interaction to create an intelligent appliance assistant.

## Overview

The system uses a custom YOLOv8 model trained on a self-collected dataset of household appliances to perform real-time object detection via webcam. Once a device is identified and locked, users can interact with the assistant through natural language — either typed or spoken — and receive contextually accurate answers generated by a Groq-hosted LLaMA 3.3 model.

The wake word "hey uniq" enables hands-free device locking and querying, making the system suitable for accessibility and hands-free operation scenarios.

## Architecture
Webcam Feed
↓
YOLOv8 (Custom Trained) — Real-time object detection at inference
↓
Device Locking — keyboard press ('l') or wake word ("hey uniq, lock this")
↓
Natural Language Query — text input or speech recognition (Google Speech API)
↓
Groq LLaMA 3.3 — LLM generates contextual answer about detected appliance
↓
pyttsx3 TTS — answer spoken back to user
## Key Components

**Custom Dataset and YOLOv8 Training**
A custom image dataset was collected and annotated for four appliance classes: Mouse, Keyboard, Food Processor, and Water Heater. YOLOv8 was fine-tuned on this dataset to achieve reliable real-time detection under varying lighting and orientations.

**Wake Word Voice Interface**
A background thread continuously listens for the wake phrase "hey uniq" using Google Speech Recognition API. Upon detection combined with the "lock" keyword, the currently visible appliance is locked for the Q&A session enabling completely hands-free operation.

**LLM-Powered Q&A**
Questions are routed to Groq LLaMA 3.3 70B model with device-specific context injection. This allows the system to answer arbitrary natural language questions about any detected appliance without being limited to a predefined set of responses.

**Streamlit Web Interface**
A Streamlit frontend provides a camera widget for snapshot-based detection, confidence-thresholded locking, and a text input for typed queries complementing the terminal-based voice flow.

## Tech Stack

| Component | Technology |
|---|---|
| Object Detection | YOLOv8 (Ultralytics, custom trained) |

## Setup

```bash
git clone https://github.com/siddlpskmi/Smart-Appliance-Assistant.git
cd Smart-Appliance-Assistant
pip install -r requirements.txt
```

Create a `.env` file:
GROQ_API_KEY=your_groq_api_key
Run the Streamlit interface:
```bash
streamlit run app.py
```

Run the full voice and webcam terminal interface:
```bash
python smart_assistant.py
```
